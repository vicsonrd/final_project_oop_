# 🌻 Valentine’s Proposal Project

## Overview
This project is a Python app that uses **Tkinter** for a Valentine’s proposal window and **Turtle Graphics** to draw a sunflower with a small heart in the center.  
It was made to show the four pillars of **Object-Oriented Programming (OOP)**.

## OOP Concepts
- **Inheritance** → Classes inherit from base abstract classes.
- **Encapsulation** → Private attributes with getters/setters.
- **Abstraction** → Abstract base classes (`ProposalApp`, `Shape`).
- **Polymorphism** → Different classes implement `draw()` or `clicked()` in unique ways.
